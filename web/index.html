DHU Data Structures Project

上海地铁路径规划系统

当前版本基于 C++ 邻接表建模与状态扩展 Dijkstra / Yen K 短路算法实现。 这个静态页面用于展示项目能力、已知现状和路径输出格式，也作为前端入口壳，确保页面本身可独立正常运行。

523 站点记录
1280+ 边数据行
3 K 候选路径输出

当前状态

  • 后端现状控制台菜单和路径规划功能可编译运行，但项目中仍存在命名混乱、说明文档滞后、部分实现冗余的问题。
  • 前端现状原工作区中并不存在真实的 `web/index.html`，说明前端入口曾经缺失或未同步。
  • 这次修复目标先保证项目结构、命名、注释和前端入口都能自洽，再逐步清理功能细节问题。

项目能力概览

  • 站点状态管理支持 CSV 批量更新、手工关闭/开启、恢复初始状态、关闭站点展示。
  • 路径规划支持单条最短时间、单条最少换乘、前 3 条候选路径。
  • 格式化输出支持 `起点[线路]`、分段路径、站内换乘提示、总耗时与站数统计。
  • 特殊线路处理4 号线支持 `内圈 / 外圈` 标记，起点虚拟换乘不计入换乘次数。
